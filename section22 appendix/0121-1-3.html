모바일 웹&앱

모바일 웹앱은 일반적인 웹사이트보다 모바일에 더 최적화 되고 네이티브 애플리케이션화된 형태를 한정하는 표현입니다.

웹 기술만을 사용해서 풀스크린모드, 애니메이션 효과, 터치 상호작용등 향상된 스타일등으 구현하여 모바일 환경에서 네이티브 애플리케이션과 유사한 실행 환경, 사용자 경험을 제공하는 형태의 애플리케이션으로 사용자를 고려합니다.

더욱 섬세하고 치밀하게 계산된 UX / UI 디자인, 모바일 환경에 최적화된 웹과 앱의 융합을 통해 실무에 적용할 수 있도록 UX / UI 최신 모바일 트렌드를 중심으로 교육합니다.
